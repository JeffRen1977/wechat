# 用 RL 做期权对冲：RLOP 把 2020 年那种极端亏损概率降到 0.33

## 核心观点
- 康奈尔/UT Austin/NTU 团队提出两种“短缺概率优先”的对冲策略：改进版 QLBS 与全新 RLOP（arXiv:2603.06587，2026.02.01）。
- 相比传统 Black-Scholes、跳扩散 (JD) 或随机波动 (SV) 模型，RLOP 在 2020 Q1 这类压力区间显著降低尾部亏损。
- RL 策略默认“少交易、先活下来”，在风险–成本平面上常落在低成本低波动的“左下角”。

## 两个新框架怎么做
1. **修改 QLBS**：在原有 Q-learning 版 Black–Scholes 中把奖励函数改成“短缺概率 + 成本”，平衡跟踪误差与交易开销。
2. **RLOP（Replication Learning of Option Pricing）**：直接以“对冲失败频率”作为目标，让策略在 delta 调整时优先避免净 PnL < 0 的情况。
3. **真实标的验证**：选取 SPY（指数）与 XOP（能源 ETF）真实上市期权，按 14/28/56 天到期、平值与轻微价外分类，在 2020Q1 与 2025Q2 等不同市场运行路径回测。
4. **评估指标**：分布性指标（净 PnL CDF、短缺概率、ES5/ES10）、静态拟合（IVRMSE）、风险–成本图（交易成本 vs. 复制误差）。

## 关键结果
- **尾部风险**：在 28 天平值 SPY 2020Q1 场景，RLOP 把短缺概率降到 0.91（传统 BS 追求的是 8.787% 的 ES5%），在 XOP 同期更是把短缺概率压到 0.78 且 ES5% = 0.502，相当于把极端亏损频率从两位数拉到个位数。
- **压力场景优势明显**：八个切片中，RL 策略在五个场景拿到最低 ES5%；在能源板块（XOP 2020Q1）平值与价外两个 moneyness 上，RLOP 同时在 ES5%、ES10% 和短缺概率上领先。
- **成本更低**：风险–成本地图显示 RL 策略总能把交易成本 E[TCT] 压到最低；在不少切片中，它们还以更小的复制 RMSE 达到同等甚至更优的尾部表现。
- **静态拟合不是一切**：即便 JD/SV 在 2020Q1 的 IVRMSE 最低，RLOP 依旧能在实际对冲路径上跑赢，说明“看得准曲面”不代表“挣得到钱”。

## 实盘启示
1. **风险指标要换**：手里只有 Delta tracking error 不够，短缺概率、期末 net CDF 更能反映 AI 交易系统的真实表现。
2. **策略组合**：QLBS 更像“成本稳压器”，RLOP 更像“极端风险守门员”。资产管理人可根据保证金/库存压力在两者之间动态切换。
3. **摩擦建模优先**：RLOP 通过将交易成本写进 RL 环，天然减少高频调仓，有利于在高手续费或限价环境运行。
4. **压力测试要看 2020**：文章用 2020Q1 验证 RLOP 的 tail 选择优势，提示我们在自测 AI 策略时别只看常态时期的 summer slice。

## 我们怎么看
随着券商、资产管理人搭建多智能体量化系统，“会算价”与“会对冲”必须拆开看。RLOP/QLBS 把奖励函数改成“概率思维”，对国内期权做市、结构性产品仓位管理都有借鉴意义。下一步值得观察的是：在中国市场这类流动性相对有限、交易成本更敏感的环境，短缺概率驱动的 RL 是否能进一步降低保证金占用和尾部 VaR。
