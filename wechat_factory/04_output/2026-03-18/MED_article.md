# 测试流水线：这是一篇用于检查公众号上传流程的文章

## 目的

这是一篇 **测试用** 医疗 AI 文章，用来验证：

1. Markdown 写入 `04_output/2026-03-18/MED_article.md`；
2. `run-gemini-images.sh` 能为该文章生成 **封面 + 配图**；
3. `wechat-draft-upload.sh` 能将正文与图片一起上传到 **公众号草稿箱**；
4. 正文中的配图引用在公众号中正常展示。

- 把 HTML 中的 `src` 替换为微信返回的在线图片 URL。

## 结语

如果你在公众号后台的 **草稿箱** 中看到这篇文章，且正文中成功显示配图，则说明：

- Markdown → HTML → 公众号 的排版链路正常；
- 封面图与正文配图都可通过 API 自动上传与替换。
