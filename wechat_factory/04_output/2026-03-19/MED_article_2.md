# MRI 联邦学习要落地？先把“谁的数据在哪”说清楚

## 论文要点
- Surrey 大学硕士论文（arXiv:2603.15901，2026.03.16）聚焦阿尔茨海默症 3D MRI 分类，提出两大实务缺口：
  1. **分区不真实**：多数联邦学习实验随机构建客户端，无法反映医院级数据鸿沟；
  2. **隐私噪声死板**：固定噪声的本地差分隐私 (Local DP) 会毁掉训练稳定性。
- 方案：
  - **Site-aware partition**：以真实“机构边界”划分数据，模拟多院协作时的样本分布差异；
  - **Adaptive Local Differential Privacy (ALDP)**：根据训练进度和参数敏感度，动态调节噪声尺度。

## 关键实验
- 数据：ADNI 3D MRI，按机构拆分 2～5 个联邦客户端。
- 优化：FedAvg、FedProx 等算法对比；FedProx 在强异质性下最稳。
- 结果：
  - **两客户端场景**下，ALDP + FedProx 达到 **80.4% 准确率**，比固定噪声 Local DP 高 5–7 个百分点；
  - 噪声调节后，loss 曲线收敛平滑，避免“差分隐私=无法训练”的传统误解；
  - 多客户端（≥4）时，Site-aware 切分暴露出真实世界的 class imbalance，但 ALDP 仍能保持接近集中式训练的表现。

## 实务启示
1. **评估协议要像真实医院**：别再用随机分片。需要公开“按医院、按设备、按成像协议”划分的基准，测试算法在极端不平衡下的稳健性。
2. **隐私预算不是一次设死**：ALDP 展示了“训练前期噪声大、后期放小”的策略，可在保证 (ε, δ) 的同时，提高可用性。
3. **FedProx 优于 FedAvg**：在跨院协作的 MRI 项目中，先选择能处理非 IID 数据的优化器，再讨论模型结构。
4. **发布代码与追踪**：作者释放 GitHub 仓库与 wandb 记录，为医疗 AI 提供了可复现实验模板。

## 我们的建议
- **医院联盟应制定“数据拓扑图”**：列出每家医院拥有的模态、数量、质控情况，以便研究者模拟真实 partition。
- **考虑“隐私自适应”作为产品功能**：例如在 SaaS 联邦平台中，对小样本站点自动提高噪声，对大站点降低噪声，达成整体最优。
- **监管沟通更有底气**：有了 Site-aware + ALDP 的实证，向 IRB / 合规部门解释“联邦模型既保护隐私又有效”会更有说服力。
