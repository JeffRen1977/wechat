# GPU狂飙之后，CPU如何接棒？

周一美股收盘，AMD 再刷 52 周新高，股价最高触及 469.22 美元；同一晚，Qualcomm 飙升 8%+，被市场当成「AI 第二梯队」里最干脆的赢家。[GF Securities 最新研报](https://seekingalpha.com/news/4590470-amd-intel-qualcomm-likely-to-benefit-from-server-cpu-super-cycle-gf-says)把这股热度概括为「服务器 CPU 超周期」：训练时代的 GPU 狂飙已成共识，而推理和 Agent 化时代需要的是更聪明、更省电的 CPU 中枢。

## 超周期的底层逻辑

GF 研报给出的数字很直白：**服务器 CPU 市场规模有望从 2025 年的 260 亿美元飙到 2030 年的 1350 亿美元，年复合增速 38%**，而且增长点集中在推理、边缘节点和混合云场景。[CryptoRank 的跟进报道](https://cryptorank.io/news/feed/fefae-amd-stock-hits-new-high-as-wall-street-bets-on-ai-cpu-supercycle)也提到，华尔街正把注意力从「谁的 GPU 最快」转向「谁能把 AI 算力调度得更稳」——这正是 CPU 的角色。

> **CPU 成了 AI 基建设计里的「乐团指挥」**：当大模型走向 Agent、走向端到端业务时，算力编排、内存读写、网络 I/O 需要一个能理解多线程任务的中枢，而不只是堆更多 GPU。

这种需求也让资本故事更有层次。Mizuho、BofA 等机构在上周就提醒客户：「AI 数据中心不再是 Nvidia 的独角戏，内存、网络、光模块、CPU 都要重估。」同一份 CryptoRank 报告还引述 BTIG 的警示——半导体板块估值已隐约复刻 1999 年的走势，25%～30% 的回撤风险随时可能敲门。

## 三家被点名的剧本

**AMD：增量赛道在 EPYC 与 MI 系列的绑定**。GF 预计 AMD 服务器 CPU 业务在 2026 年可能再冲 73% 增长，主因是第四代 EPYC 已在云厂、超算和 AI 推理节点里形成「带宽 + 内存」优势。AMD 最新财报也强调，数据中心业务已经是营收和利润的首要驱动，未来三到五年的 CPU 市场增速预期从 18% 提到 35%。

**Intel：靠制程自救，也靠 IFS 打开代工客户**。GF 把 Intel 数据中心与 AI Segment 的 2026 年增速预估上修至 39%，理由是：一方面，Granite Rapids、Sierra Forest 这代产品切入了能效敏感的云原生负载；另一方面，IFS 接到的代工订单（含自研 14A/18A）能摊薄资本开支，让自家 CPU 的毛利表现不再「垫底」。

**Qualcomm：赌的是「低功耗 + ARM」的下一幕**。MarketBeat 汇总的多家机构观点显示，Qualcomm 股价大涨不仅因为手机回暖，更在于它把 Nuvia 团队的 ARM CPU 推向数据中心、PC 与车载的动作终于明牌。[Daiwa 刚把评级调到 Outperform](https://www.marketbeat.com/stocks/NASDAQ/QCOM/news/)，6 月 24 日的投资者日被视为验证点：能否交出服务器样机、能否讲清楚与微软 Copilot+ PC 同步的云端故事。

GF 还点名 ARM 阵营的新变量：亚马逊 Graviton、Google Axion、Nvidia Grace 的自研节奏明显加速。换句话说，x86 双雄虽然暂时稳坐 90% 市占，但必须在能效和生态上回应这些轻量级挑战。

## 普通投资者要盯什么信号

**第一，盯数据中心资本开支的结构**。Meta、Microsoft、Amazon 在最新财报里都把 2024～2025 的 CAPEX 指引往上调，但细分里「AI 服务器」与「通用服务器」的预算比例才是 CPU 需求的真实指针。

**第二，关注推理端口的功耗指标**。就像手机时代的功耗墙，总算力不是唯一 KPI。谁能在 1U 甚至 Edge Box 里把 TCO 压下来，谁就更可能抢到推理/Agent 场景的话语权。

**第三，把库存周期和政策信号都纳入雷达**。BTIG 的估值警示并非危言耸听，回想 2022 年的芯片熊市，真正引爆调整的是「库存 + 宏观」的双杀。现在美债收益率仍高，任何政策或需求侧的闪失都会把超周期故事打回起点。
