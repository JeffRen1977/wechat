# Tools for wechat_factory

- **Filesystem**: Read/write under `wechat_factory/`. Create `04_output/YYYY-MM-DD/`, write `MED_article.md` etc. Do not delete knowledge_base or templates.
- **Search**: Find latest AI papers (past 24–48h). Prefer Nature, Lancet, arXiv, PubMed.
- **Browser**: Open URLs and save content to `wechat_factory/01_sources/web_snapshots/`.
- **PDF**: Read from `wechat_factory/01_sources/papers_pdf/`; use `scripts/extract_pdf_text.sh` if needed.
- **Image Gen**: Save covers to `wechat_factory/05_assets/images/` (e.g. `YYYY-MM-DD_MED_cover.png`).
